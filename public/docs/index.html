Introduction

This is the API documentation for Edu Tech API.

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Assignments

Display a listing of the assignments.

This endpoint retrieves all assignments.

Example request:
curl --request GET \
    --get "http://localhost/api/assignments" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/assignments"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Store a newly created assignment.

This endpoint allows you to create a new assignment.

Example request:
curl --request POST \
    "http://localhost/api/assignments" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"title\": \"Homework #1\",
    \"description\": \"Solve problems 1-10 from the textbook.\",
    \"course_id\": 2,
    \"due_date\": \"2024-10-20\"
}"

const url = new URL(
    "http://localhost/api/assignments"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "title": "Homework #1",
    "description": "Solve problems 1-10 from the textbook.",
    "course_id": 2,
    "due_date": "2024-10-20"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (201):


{
    "id": 1,

Accept      

Example: application/json

Body Parameters

The name of the user. Example: John Doe

password_confirmation   string
